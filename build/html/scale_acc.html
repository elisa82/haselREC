haselrec.scale_acc.scale_acc(n_gm, nga, path_nga, path_esm, source, event, station, name, output_folder, sf)[source]¶

Scales selected accelerograms and writes them in the output folder.

nGM x 2 files are created. Their name is:

GMR_time_scaled_acc_<GMnum>_<comp>.txt
where:
  • <GMnum> is a sequential number ranging from 1 to nGM
  • <comp> can be 1 or 2 and indicates the horizontal component of motion

Each file contains the selected scaled accelerograms, expressed with 2 columns: time series (s) and accelerations (g)
